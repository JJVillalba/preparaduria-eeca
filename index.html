{2x+6y=0−3x−9y=18.\begin{cases}2x + 6y&&= 0 \\ -3x - 9y&&= 18.\end{cases}{2x+6y−3x−9y​​=0=18.​

Solución:

Graficando las ecuaciones podemos observar que estas representan rectas paralelas, por lo que el sistema de ecuaciones no tienen solución verificaremos este echo algebraicamente despejando para los valores de xxx e yyy.

{Grafico de las relaciones $-3x-9y=18$ y $2x+6y=0$.+}

Supongamos ahora que existe una solución al sistema de ecuaciones y despejemos xxx en función a yyy obtenemos para obtener que

2x+6y=0  ⟺  x=−3y,2x + 6y = 0 \iff x = -3y,2x+6y=0⟺x=−3y,

Sustituyendo en la segunda ecuación obtenemos que:

−3(−3y)−9y=18  ⟺  9y−9y=18  ⟺  0=18.-3(-3y) - 9y = 18 \iff 9y - 9y = 18 \iff 0 = 18.−3(−3y)−9y=18⟺9y−9y=18⟺0=18.

Que evidentemente es una contradicción, que provino de asumir que nuestro sistema de ecuaciones tenia solución.
